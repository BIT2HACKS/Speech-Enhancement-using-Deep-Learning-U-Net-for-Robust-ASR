This project implements a deep learning-based speech enhancement system using a U-Net architecture. The model is trained to reconstruct clean speech from noisy audio signals by operating on Mel spectrogram representations.

Noisy speech is generated by adding synthetic noise to clean audio samples from the LJSpeech dataset. The model learns to map noisy spectrograms to clean spectrograms, and the enhanced audio is reconstructed using inverse Mel scaling and Griffin-Lim algorithm.

The system demonstrates effective noise reduction and improved speech quality, validated through both qualitative (audio listening) and quantitative (SNR improvement) evaluation.
